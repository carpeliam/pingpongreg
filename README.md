# Ping Pong Registration

Rails 5 API backend, React Redux Webpack frontend.

[Pivotal Tracker Integration](https://www.pivotaltracker.com/n/projects/1615201).

## Build

### Back End
```
cd backend
bundle install
```

### Front End

To export frontend into Rails backend:

```
cd frontend
npm install
npm run build
```


## Test

### Back End
```
rake test
```

### Front End
```
npm run test
```
